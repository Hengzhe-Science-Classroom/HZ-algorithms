## Features

- **Interactive visualizations** — sorting animators, graph editors, DP table fillers, flow network simulators, all on Canvas 2D
- **KaTeX math rendering** — rigorous notation throughout (Big-O, recurrences, proofs)
- **Exercises with hints & solutions** — from tracing algorithms to proving correctness and complexity
- **Progress tracking** — saved locally in your browser
- **Zero dependencies** — just open the link and learn

## References

- Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest & Clifford Stein, *Introduction to Algorithms* (4th ed., MIT Press, 2022)
- Jon Kleinberg & Éva Tardos, *Algorithm Design* (Pearson, 2006)
- Jeff Erickson, *Algorithms* (self-published, 2019; [free online](http://jeffe.cs.illinois.edu/teaching/algorithms/))
